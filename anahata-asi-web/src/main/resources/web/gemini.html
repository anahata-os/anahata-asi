Anahata Gemini

The bridge between the Anahata Core and Google's GenAI ecosystem.

The Gemini module acts as a high-performance adapter, translating Anahata's domain model into native Google GenAI requests and responses.

Key Components

  • GeminiAgiProvider: Concrete implementation of the provider interface.
  • GeminiModel: A native wrapper around Google's GenerativeModel.
  • GeminiContentAdapter: Strategic logic for multi-modal part translation.
