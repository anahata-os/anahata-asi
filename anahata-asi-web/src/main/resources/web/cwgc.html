CWGC Protocol

Context Window Garbage Collection • V2 Standard
The Core Philosophy: Transparency and Stability. The system enforces an Atomic Interaction model: conversations remain structurally intact until a turn has completely expired.

1. The PruningState Enum

Both AbstractMessage and AbstractPart implement an explicit state-based control system:

State Enum Value Behavior Immunity
Auto AUTO Default state. Governed by remainingDepth countdown. No
Pruned PRUNED Explicitly hidden from the model. Content removed, metadata hints preserved. No
Pinned PINNED Always visible to the model. Ignores remainingDepth. YES

2. Atomic Interaction Lifecycle

Atomic Interaction Model

To prevent "mutilated messages," interactions remain in history as "ghosts" (containing effectively pruned metadata) until every part in the message has expired.

The Rule: A message is only garbage collected when ALL sibling parts are collectable.

Surgical Removal

Heavy content (Blobs) and internal reasoning (Thoughts) are eligible for immediate hard removal upon expiration to minimize memory and token bloat, even if their parent interaction is still active.

3. Pruning Logic

isEffectivelyPruned()

This derived boolean determines if content is excluded from the AI prompt. A part is effectively pruned if:

  • It is explicitly set to PRUNED.
  • Time-out: Its state is AUTO and its remainingDepth <= 0.

Note: PINNED state always overrides expiration logic. A Message is effectively pruned only if all its parts are invisible.

4. The "Autumn Leaf" Metaphor

Active (Green)

Representing the AUTO state with positive remaining depth. These are full of life and context.

Withered (Light Brown)

Representing AUTO parts that have expired. They are "Effectively Pruned" and ready to fall from the tree.

Dead (Dark Brown)

Representing PRUNED parts. These have been explicitly cut off from context by the user or model.

Tool Call Stability: Tool calls remain as metadata ghosts in context long after their results are pruned, ensuring the model maintains a coherent audit trail of its actions.
