Anahata Swing

High-fidelity, provider-agnostic UI components for building agentic workflows in Java.

The AgiPanel

The primary container for an AGI session. It coordinates the chat interface, resource management, and real-time status monitoring in a unified, responsive layout.

Diff-and-Update Strategy

To maintain extreme performance in conversations with thousands of turns, Anahata Swing uses a surgical "Diff-and-Update" rendering engine:

  • Message Level: ConversationPanel tracks AbstractMessage objects and reuses panel instances.
  • Part Level: Individual message parts are rendered using specialized, reusable PartPanels.
  • Text Level: Markdown is parsed into descriptors and rendered via high-performance segments.
