NetBeans ASI Studio

Deep IDE integration for the ultimate Java coding experience.

The NetBeans module transforms the IDE into a high-fidelity ASI Container, giving the model direct access to project structures, compilation errors, and native refactoring APIs.

Annotator Strategy

We use a non-intrusive annotation system to provide context within the IDE:

  • Project Icons: Real-time badges for Anahata-enabled projects.
  • Context Menus: Unified "AI Context" submenus across all file types.
  • Editor Feedback: Direct line annotations from the ASI on source code.
