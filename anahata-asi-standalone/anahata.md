# Anahata ASI Standalone (`anahata-asi-standalone`)

> [!IMPORTANT]
> This file is an extension of the `anahata.md` in the parent project. Always keep the root `anahata.md` in context as it contains the master Coding Principles and Javadoc Standards.

## 1. Purpose
Provides standalone Java launchers for both CLI and Swing interfaces outside of an IDE.

## 2. Execution
Invoke the NetBeans project action "run" or use the Maven Shade generated JAR.

## 3. Session Persistence
Sessions are stored in: `~/.anahata/asi/standalone/sessions`
